빠른 사용 가이드

  1. 📹 테스트 영상 선택 드롭다운에서 보고 싶은 영상을 고릅니다.
  2. 영상 로드 버튼을 눌러 플레이어에 영상을 불러옵니다.
  3. 비디오 화면의 ▶︎ 재생 버튼을 클릭하면 곧바로 재생이 시작됩니다.
  4. 화질·버퍼 등 재생 품질은 오른쪽 컨트롤 패널에서 자유롭게 조정해 보세요.
  5. 🌐 네트워크 상태 시뮬레이션
      ㆍ아래 슬라이더로 대역폭 제한을 설정하면 실제 네트워크가 느린 것처럼 테스트할 수 있습니다.
      ㆍ네트워크 테스트 버튼을 누르면 4 초 간격으로 대역폭이 자동 변경되어
        ABR(적응형 비트레이트) 반응을 한눈에 확인할 수 있습니다.
      ㆍ“제한 해제” 버튼을 누르면 언제든 원래 상태로 복원됩니다.
